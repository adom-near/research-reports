重灌後第一步
Step 1. 安裝 Homebrew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
Step 2. 核心開發工具
brew install node ffmpeg gh tmux testdisk python3
Step 3. 安裝 Claude Code
npm install -g @anthropic-ai/claude-code ccusage
Step 4. 登入 Claude Code
claude
會跳轉瀏覽器登入 Anthropic 帳號。
Step 5. 恢復 Claude 記憶
mkdir -p ~/.claude/projects/-Users-wt-PyCharmMiscProject/memory/
tar xzf claude_memory_backup_20260329.tar.gz -C ~/.claude/projects/-Users-wt-PyCharmMiscProject/memory/
Step 6. 恢復 DaVinci Resolve 專案
先從官網下載安裝 DaVinci Resolve，然後：
mkdir -p ~/Library/Application\ Support/Blackmagic\ Design/
tar xzf DaVinci_Resolve_backup_20260327.tar.gz -C ~/Library/Application\ Support/Blackmagic\ Design/
Step 7. Clone 專案
mkdir -p ~/PyCharmMiscProject
cd ~/PyCharmMiscProject
git clone https://github.com/adom-near/research-reports.git 研究報告網站
Step 8. Python 套件
pip3 install edge-tts
Step 9. 啟動 Discord 連線
cd ~/PyCharmMiscProject
claude --channels plugin:discord@claude-plugins-official
