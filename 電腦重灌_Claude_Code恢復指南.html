Claude Code 完整恢復指南

這份指南涵蓋電腦重灌後恢復完整 Claude Code 工作環境的所有步驟。每個項目都可以打勾追蹤進度，進度會自動儲存在瀏覽器中。

<\!-- Section 1 -->
1
前置準備（重灌前）
<\!-- Section 2 -->
2
安裝基礎工具
2-1. 安裝 Homebrew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
2-2. 安裝核心工具
brew install node git python3 ffmpeg
2-3. 安裝 PyCharm
從官網下載 PyCharm Community Edition 並安裝。
<\!-- Section 3 -->
3
安裝 Claude Code
npm install -g @anthropic-ai/claude-code
第一次執行 claude 會要求登入 Anthropic 帳號。
<\!-- Section 4 -->
4
恢復工作區
cd ~
git clone https://github.com/你的帳號/PyCharmMiscProject.git
cd PyCharmMiscProject
CLAUDE.md 在 repo 裡，clone 下來就有，不需要額外設定。
<\!-- Section 5 -->
5
恢復 Claude 記憶
mkdir -p ~/.claude/projects/-Users-wt-PyCharmMiscProject/memory/
cd ~/.claude/projects/-Users-wt-PyCharmMiscProject/memory/
tar xzf /path/to/claude_memory_backup_20260327.tar.gz
路徑中的 -Users-wt-PyCharmMiscProject 要跟實際工作目錄對應。如果使用者名稱或路徑不同，需要調整。例如使用者改叫 john，路徑就是 -Users-john-PyCharmMiscProject。
<\!-- Section 6 -->
6
設定 Discord Plugin
6-1. 啟動 Claude Code 並載入 Discord
cd ~/PyCharmMiscProject
claude --channels plugin:discord@claude-plugins-official
6-2. 配對 Discord Bot
在 Claude Code 裡執行 /discord:access 進行 bot 配對。
6-3. 恢復 access.json（如果有備份）
cp /path/to/access.json ~/.claude/channels/discord/access.json
如果有備份 access.json，可以直接複製回去跳過配對步驟。
<\!-- Section 7 -->
7
恢復 DaVinci Resolve
7-1. 安裝 DaVinci Resolve
從官網下載安裝 DaVinci Resolve。
7-2. 恢復專案資料庫
mkdir -p ~/Library/Application\ Support/Blackmagic\ Design/
tar xzf /path/to/DaVinci_Resolve_backup_20260327.tar.gz \
  -C ~/Library/Application\ Support/Blackmagic\ Design/
<\!-- Section 8 -->
8
安裝其他常用工具
8-1. Python 套件
pip3 install edge-tts playwright
8-2. Playwright 瀏覽器
python3 -m playwright install
8-3. 資料救援工具
brew install testdisk
8-4. Remotion 專案依賴（如果需要）
cd ~/PyCharmMiscProject/高市圖動畫 && npm install
cd ~/PyCharmMiscProject/Ai影片生成/remotion-test && npm install
<\!-- Section 9 -->
9
GitHub Pages 研究報告網站
cd ~/PyCharmMiscProject/研究報告網站
git remote -v  # 確認 remote 正確
網站會自動從 GitHub Pages 部署，不需要額外設定。
<\!-- Section 10 -->
10
備用電腦設定（Discord 不斷線方案）
如果要在備用電腦上跑 Claude Code 維持 Discord 連線：
10-1. 完成基礎設定
在備用電腦上完成 Section 2 ~ 6 的步驟。
10-2. 啟動 Discord 連線
開一個 Terminal 視窗執行，不要關閉這個視窗：
cd ~/PyCharmMiscProject
claude --channels plugin:discord@claude-plugins-official
10-3. 防止電腦睡眠
caffeinate -d &
10-4. 雙機並行
主電腦有需要時另外開 Claude Code session（不帶 --channels），兩邊不衝突。
